عنوان:
کانکتور USB Type-C نری (Plug) به همراه کاور سفید

قیمت:
165,000

توضیحات کوتاه:
کانکتور تعمیری نری Type-C USB با قابلیت انتقال دیتا

توضیحات کامل:
معرفی

کانکتور تعمیری نری Type C که به آن سوکت تعمیری شارژر اندروید نیز می گویند. سر شارژر های Type C بعد از مدتی استفاده خراب شده و یا کنده می شوند. کابل شارژ های فابریک گوشی تلفن همراه نیز قیمت های بالایی دارند. زمانی که کابل شارژ تلفن همراه شما آسیب می بیند به جای خرید یک کابل شارژ جدید می توانید سری آن را با یک سری تعمیری سری type C تعویض نمایید.

سری شارژر تیپ C دارای چهار قسمت است؛ قسمت اول سری شارژر که دارای یک برد کوچک می باشد. قسمت دوم یک کاور که از سیم های لحیم شده و همچنین مدارات داخل کانکتور نری شارژر را محافظت می کند. قسمت سوم یک سیم گیر بوده که باعث می شود سیم شارژر بهتر و منظم تر متصل شود. قسمت چهارم یک گیره که در جلو کانکتور برای محکم شدن قرار داده می شود.
