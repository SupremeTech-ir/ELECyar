Introduction

This is a Male Micro USB Socket With Plastic Shell (Micro USB B Male Connector )  which each set consists a 4 pins sockets (connector body with solderable pins), a plastic shell, and 2 supporters or strain relief (one for wire and one for socket in the shell). Micro USB type B connector is widely used in mobile applications and chargers. This connector is matched with 24AWG wires and a smaller size. You can create your favorite adapter and cable for the smartphone charging cable, data transfer cables, charging adapters, and so on, also can repair existing smartphone cables.
