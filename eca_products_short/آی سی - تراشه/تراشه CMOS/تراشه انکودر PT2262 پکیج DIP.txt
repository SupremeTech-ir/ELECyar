Description

PT2262 is a remote control encoder paired with PT2272 utilizing CMOS Technology. It encodes data
and address pins into a serial coded waveform suitable for RF or IR modulation. PT2262 has a
maximum of 12 bits of tri-state address pins providing up to 531,441 (or 312) address codes; thereby,
drastically reducing any code collision and unauthorized code scanning possibilities.

Features

CMOS Technology
Low Power Consumption
Very High Noise Immunity
Up to 12 Tri-State Code Address Pins
Up to 6 Data Pins
Wide Range of Operating Voltage: Vcc = 4 ~ 15 Volts
Single Resistor Oscillator
Latch or Momentary Output Type
Available in DIP and SOP
